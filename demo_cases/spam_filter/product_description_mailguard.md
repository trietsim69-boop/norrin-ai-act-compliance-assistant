# MailGuard — Email Spam Filter

**Product type:** Email security add-on  
**Deployment:** On-premise gateway appliance

MailGuard is an **email spam filter** that classifies incoming messages as spam, phishing, or legitimate mail. It protects corporate inboxes before messages reach user mail clients.

## Function

- Scans headers, body text, and attachment metadata.
- Assigns a spam probability score using a trained ML classifier.
- Quarantines messages above the configured threshold.
- Users can mark quarantined messages as "not spam" to improve local tuning.

## Impact on individuals

- Narrow filtering task; does not evaluate individuals' legal rights or creditworthiness.
- Misclassification may delay legitimate mail but does not determine legal rights or access to essential services.

## Regulatory context

Internal security team considers this **low-risk infrastructure** with no AI Act high-risk mapping identified to date.
