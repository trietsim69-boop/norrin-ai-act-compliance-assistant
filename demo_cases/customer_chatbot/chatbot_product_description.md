# SupportBot — Customer Support Chatbot

**Product:** SupportBot v3  
**Owner:** Customer Experience team

SupportBot is a **customer support chatbot** that interacts with end users on our website and mobile app. It answers FAQs, tracks orders, and escalates complex issues to human agents when confidence is low.

## Interaction model

- Users type natural-language questions in a chat widget.
- The bot responds in real time with policy-compliant answers.
- Conversational tone; supports English, German, and French.

## Technology

- Built on a **third-party large language model (LLM)** via API (OpenAI-compatible endpoint).
- Retrieval layer over our knowledge base for factual answers.
- Session logs stored for quality review.
