# TalentRank Product Description

TalentRank is an AI recruitment screening tool used by the HR team to rank job applicants for open roles.

The system receives CVs, application forms, and structured questionnaire answers. A machine learning model scores each candidate against the role requirements and produces a ranked shortlist for recruiters.

The output is a candidate score, a shortlist position, and a short explanation of matching skills. Recruiters use the shortlist to decide which applicants move to interview.
