# TalentRank AI — Product Description

**Vendor:** Norrin HR Tech  
**Version:** 2.1 (pilot)

TalentRank is an AI-powered recruitment platform that scores and ranks job applicants from uploaded CVs, cover letters, and structured screening questionnaires. The system produces a ranked shortlist for human recruiters.

## Core capabilities

- Parses CVs and extracts skills, experience, and education fields using a fine-tuned language model.
- Computes a compatibility score against the job description and role requirements.
- Ranks candidates from highest to lowest predicted fit.
- Generates a one-page recruiter brief summarising strengths and gaps for each shortlisted applicant.

## Intended users

HR teams and hiring managers in mid-to-large enterprises running high-volume recruitment campaigns.

## Deployment

Cloud SaaS. Customer HR systems push applicant data via API. Recruiters access the ranked shortlist in a web dashboard.
